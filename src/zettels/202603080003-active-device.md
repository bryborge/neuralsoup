# Active Device

## Core Concept

> In electronics, an active device is a circuit component that requires an external power source in order to function.

## Explanation

An active device is one that can produce signal gain and control electron flow. 

## Example

- Transistors
- Integrated Circuits
- Diodes
- Sources
- Optoelectronic Devices
