# Linear Equation

## Core Concept

> A linear equation is an algebraic equation of the form:
$$
c_1x_1 + c_2x_2 + \space ... \space + c_nx_n
$$

Where $c_n$ is any constant and $x_n$ is any variable raised to the first power.

## Explanation

On a Cartesian coordinate plane, a linear equation represents a straight line with a constant slope.

## Examples

- Slope-Intercept form: $y = mx + b$
- Standard form: $Ax + By = C$
- Point-Slope form: $y - y_1 = m(x - x_1)$
