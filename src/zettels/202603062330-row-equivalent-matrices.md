# Row-Equivalent Matrices

## Core Concept

>  Two matrices are row equivalent if, and only if, they have the same [[202603062340-reduced-row-echelon-form|reduced row echelon form (RREF)]].  Two matrices representing linear systems are also said to be row equivalent when they have the same [[202603061536-solution-set|solution set]].
