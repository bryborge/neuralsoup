# Ordinary Node

## Core Concept

> An ordinary node is a point in an electrical circuit where 2 electrical devices connect with each other.

```tikz
\usepackage{circuitikz}
\begin{document}
\begin{tikzpicture}[american]
  \coordinate (N1) at (0,0);
  \node[above] at (N1) {$N_1$};
  \draw (N1) to[short,-o] ++(-0.2,0) to[C,l=$C$] ++(-2.0,0) node[left] {$V_{in}$};
  \draw (N1) to[short] ++(0.2,0) to[R,l=$R$] ++(2.0,0) coordinate (afterR);
  \draw (afterR) -- ++(0,-1) node[ground] {};
\end{tikzpicture}
\end{document}
```

$N_1$ in this example is the ordinary node (where the capacitor $C$ and resistor $R$ are connected end-to-end).
