# Nodes

## Core Concept

> A node in a circuit is an electrical connection point at which 2 or more wires or components join together.

```tikz
\usepackage{circuitikz}
\begin{document}
\begin{tikzpicture}[american]
  \coordinate (N1) at (0,0);
  \node[above] at (N1) {$N_1$};
  \draw (N1) to[short,-o] ++(-0.2,0) to[R] ++(-2.0,0) node[left]{$V_{in}$};
  \draw (N1) to[short] ++(0.2,0) to[led] ++(2.0,0) to[short] ++(0,-1) node[ground]{};
\end{tikzpicture}
\end{document}
```

## Explanation

Nodes are often denoted as $N_1$, $N_2$, etc. There are two kinds of nodes -- [[202603082335-extraordinary-node|extraordinary nodes]] and [[202603082335-ordinary-node|ordinary nodes]].
