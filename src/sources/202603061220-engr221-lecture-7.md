## Key Terms & Definitions

- **Difference Amplifier:** An op-amp circuit whose output is proportional to the difference between two input voltages.
- **Differential Mode:** The difference between two input voltages (va - vb).
- **Common Mode:** The average of two input voltages.
- **Voltage Follower/Buffer:** A circuit that isolates the input source from load variations, providing unity gain (vo = vs).
- **Op-Amp (Operational Amplifier):** A high-gain voltage amplifier with differential inputs and, typically, a single-ended output.
- **Node-Voltage Method:** A circuit analysis technique using node voltages as variables.
- **Mesh Current Method:** A circuit analysis technique using mesh currents as variables.
- **Thevenin/Norton Equivalent Circuits:** Simplified representations of circuits for analysis.
- **Inverting/Noninverting Amplifier:** Op-amp configurations that invert or maintain the phase of the input signal, respectively.
- **Summing Amplifier:** An op-amp circuit that outputs the sum of multiple input voltages.
