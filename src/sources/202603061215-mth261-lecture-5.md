## Key Terms & Definitions

- **Linearly Independent:** A set of vectors where the only solution to x₁v₁ + ... + xₚvₚ = 0 is the trivial solution (all x = 0).
- **Linearly Dependent:** A set of vectors where there exists a nontrivial solution to x₁v₁ + ... + xₚvₚ = 0.
- **Trivial Solution:** The solution x = 0 in the vector equation x₁v₁ + ... + xₚvₚ = 0.
- **Nontrivial Solution:** Any solution to the vector equation where at least one x ≠ 0.
- **Linear Dependence Relation:** An equation showing a nontrivial combination of vectors summing to zero.
- **Span:** The set of all linear combinations of a set of vectors.
- **Free Variable:** A variable in a system that is not a leading variable, indicating possible dependence.
