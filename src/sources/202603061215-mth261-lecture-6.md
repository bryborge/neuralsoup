## Key Terms & Definitions

- **Transformation (Mapping):** A rule assigning each vector in $\mathbb{R}^n$ to a vector in $\mathbb{R}^m$.
- **Domain:** The set of input vectors for a transformation.
- **Codomain:** The set of possible output vectors for a transformation.
- **Range:** The set of actual outputs (images) of a transformation.
- **Matrix Transformation:** A transformation defined by T(x) = Ax for some matrix A.
- **Linear Transformation:** A transformation T satisfying T(u+v) = T(u) + T(v) and T(cu) = cT(u).
- **Superposition Principle:** T(cu + dv) = cT(u) + dT(v) for all scalars c, d and vectors u, v.
- **Contraction:** A linear transformation T(x) = rx with 0 < r < 1.
- **Dilation:** A linear transformation T(x) = rx with r > 1.
