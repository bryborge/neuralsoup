## Key Terms & Definitions

- **Basis**: A set of vectors in a vector space that is linearly independent and spans the space.
- **Coordinate vector relative to a basis**: The unique vector of weights (coordinates) such that a vector is expressed as a linear combination of the basis vectors.
- **Change-of-coordinates matrix**: A matrix that converts coordinate vectors from one basis to another.
- **Transition matrix**: Another term for the change-of-coordinates matrix, used to transition between two bases.
- **Unique representation**: The property that every vector in a vector space can be written uniquely as a linear combination of basis vectors.
