## Key Terms & Definitions

- **Eigenvalue (ω):** A scalar such that there exists a nontrivial solution x to Ax = ωx for a square matrix A.
- **Eigenvector:** A nonzero vector x that satisfies Ax = ωx for some eigenvalue ω.
- **Eigenspace (Eω):** The null space of (A - ωI), consisting of all eigenvectors corresponding to eigenvalue ω.
- **Characteristic Polynomial:** The polynomial det(A - ωI), whose roots are the eigenvalues of A.
- **Multiplicity:** The number of times an eigenvalue appears as a root of the characteristic polynomial.
- **Triangular Matrix:** A matrix where all entries above or below the main diagonal are zero; its eigenvalues are the diagonal entries.
- **Nontrivial Solution:** A solution x ≠ 0 to a matrix equation.
- **Invertible Matrix:** A matrix that has an inverse; A is not invertible if 0 is an eigenvalue.
