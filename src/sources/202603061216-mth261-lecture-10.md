## Key Terms & Definitions

- **Subspace:** A subset of Rⁿ that contains the zero vector, is closed under addition, and closed under scalar multiplication.
- **Span:** The set of all linear combinations of a set of vectors.
- **Column space (ColA):** The set of all linear combinations of the columns of a matrix A; a subspace of Rᵐ.
- **Null space (NulA):** The set of all solutions to Ax = 0; a subspace of Rⁿ.
- **Basis:** A linearly independent set of vectors that spans a subspace.
- **Dimension:** The number of vectors in a basis for a subspace.
- **Rank:** The dimension of the column space of a matrix.
- **Nullity:** The dimension of the null space of a matrix.
- **Pivot columns:** Columns in a matrix corresponding to leading ones in its echelon form; form a basis for ColA.
- **Row space (RowA):** The set of all linear combinations of the rows of a matrix.
- **Rank-nullity theorem:** For an n-column matrix A, dim NulA + rankA = n.
