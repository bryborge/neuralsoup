## Key Terms & Definitions

- **Homogeneous System:** A linear system of the form Ax = 0.
- **Trivial Solution:** The solution x = 0 to a homogeneous system.
- **Nontrivial Solution:** Any solution to Ax = 0 where x ≠ 0.
- **Free Variable:** A variable in a linear system that is not a leading variable in row-echelon form, allowing for infinite solutions.
- **Parametric Vector Form:** Expressing the general solution to a system as a vector equation with parameters.
- **Particular Solution:** A specific solution to a non-homogeneous system Ax = b.
- **General Solution:** The sum of a particular solution and the general solution to the associated homogeneous system.
- **Line in Vector Form:** v(r) = ru + w, a line parallel to u passing through w.
- **Plane in Vector Form:** A set of points described by two parameters and two direction vectors.
