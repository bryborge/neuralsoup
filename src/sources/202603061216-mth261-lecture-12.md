## Key Terms & Definitions

- **Diagonal Matrix:** A matrix with nonzero entries only on its main diagonal.
- **Diagonalization:** The process of expressing a matrix A as A = P D P⁻¹, where D is diagonal and P is invertible.
- **Diagonalizable:** A matrix is diagonalizable if it can be written as A = P D P⁻¹ for some invertible P and diagonal D.
- **Eigenvalue:** A scalar λ such that Ax = λx for some nonzero vector x.
- **Eigenvector:** A nonzero vector x satisfying Ax = λx for some eigenvalue λ.
- **Linearly Independent Eigenvectors:** A set of eigenvectors where no vector is a linear combination of the others; required for diagonalization.
- **Multiplicity:** The number of times an eigenvalue appears as a root of the characteristic polynomial.
- **Eigenspace:** The set of all eigenvectors corresponding to a particular eigenvalue, plus the zero vector.
