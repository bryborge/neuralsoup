## Key Terms & Definitions

- **Operational Amplifier (OpAmp):** A high-gain electronic voltage amplifier with differential inputs and, usually, a single-ended output.
- **Noninverting Input (+):** The OpAmp terminal where a positive input increases the output voltage.
- **Inverting Input (−):** The OpAmp terminal where a positive input decreases the output voltage.
- **Output:** The terminal where the amplified signal is produced.
- **Positive/Negative Power Supply (V+, V−):** Terminals providing the required supply voltages for OpAmp operation.
- **Negative Feedback:** A configuration where a portion of the output is fed back to the inverting input to stabilize and linearize the amplifier.
- **Saturation:** The state where the OpAmp output is limited by the supply voltages.
- **Linear Region (Dynamic Range):** The range where the OpAmp output is a linear function of the input difference.
- **Virtual Short:** The condition in an ideal OpAmp where the voltage difference between inputs is nearly zero (vp ≈ vn).
- **Inverting Amplifier:** A circuit where the output is an amplified, inverted version of the input.
- **Noninverting Amplifier:** A circuit where the output is an amplified, non-inverted version of the input.
- **Summing Amplifier:** A circuit that outputs a scaled, inverted sum of multiple input voltages.
- **Open Loop Gain (A):** The gain of the OpAmp without feedback, ideally infinite.
- **Input Resistance:** Ideally infinite for OpAmps, meaning negligible current flows into the input terminals.
- **Output Resistance:** Ideally zero for OpAmps, allowing maximum power transfer to the load.
