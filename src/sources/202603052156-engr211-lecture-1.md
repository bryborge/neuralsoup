## Key Terms & Definitions

- **Electrical Engineering:** Branch of engineering dealing with the technology of electricity.
- **Electric Circuit:** System of passive devices, voltage/current sources, and switches.
- **Electronic Circuit:** System with active devices (transistors, ICs) requiring power.
- **Passive Devices:** Components like resistors, capacitors, inductors; do not require power to function.
- **Active Devices:** Components like transistors, ICs; require power and can amplify signals.
- **Node:** Electrical connection point joining wires or components.
- **Branch:** Path between two nodes containing one element.
- **Path:** Sequence of branches with no repeated nodes.
- **Loop:** Closed path starting and ending at the same node.
- **Mesh:** Smallest loop in a circuit.
- **Series:** Elements sharing the same current.
- **Parallel:** Elements sharing the same voltage.
- **Planar Circuit:** Circuit drawable in 2D without crossing branches.
- **Nonplanar Circuit:** Circuit that cannot be redrawn without crossing branches.
- **Electric Charge:** Fundamental property, bipolar, measured in coulombs (C).
- **Current (I):** Movement of charge, measured in amperes (A).
- **Voltage (V):** Energy per unit charge, measured in volts (V).
- **Ground:** Reference point for measuring voltage in circuits.
- **Node Voltage:** Voltage at a node relative to ground.
- **Voltage Difference:** Difference in voltage between two nodes.
- **Open Circuit:** Path with a break, no current flow.
- **Short Circuit:** Path with complete continuity.
- **Power (P):** Rate of energy transfer, P = VI, measured in watts (W).
- **Ideal Voltage Source:** Provides fixed voltage regardless of load.
- **Ideal Current Source:** Maintains set current regardless of voltage.
- **Independent Source:** Source not dependent on other circuit elements.
- **Dependent Source:** Source whose value depends on another voltage or current in the circuit.
