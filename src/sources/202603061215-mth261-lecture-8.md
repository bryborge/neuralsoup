## Key Terms & Definitions

- **Matrix (m × n):** A rectangular array with m rows and n columns.
- **(i, j)-entry:** The element in the ith row and jth column of a matrix.
- **Main diagonal:** Entries a₁₁, a₂₂, ..., aₙₙ in a square matrix.
- **Square matrix:** A matrix with the same number of rows and columns.
- **Diagonal matrix:** A square matrix with zeros off the main diagonal.
- **Matrix addition/subtraction:** Entry-by-entry operation, only for matrices of the same size.
- **Scalar multiplication:** Multiplying every entry of a matrix by a scalar.
- **Matrix multiplication:** Product AB defined when columns of A = rows of B; (AB)ᵢⱼ = rowᵢ(A) · colⱼ(B).
- **Dot product:** For vectors x, y: x · y = x₁y₁ + ... + xₙyₙ.
- **Transpose (Aᵗ):** Switches rows and columns of a matrix.
- **Identity matrix (Iₙ):** Square matrix with 1s on the main diagonal, 0s elsewhere.
- **Inverse (A⁻¹):** For square matrix A, A⁻¹A = AA⁻¹ = I if A⁻¹ exists.
- **Determinant (2x2):** For A = [[a, b], [c, d]], det(A) = ad - bc.
- **Nonsingular/invertible:** A matrix with an inverse.
- **Singular:** A matrix without an inverse.
- **Row reduction:** Method to solve Ax = b or find A⁻¹ by reducing [A I] to [I A⁻¹].
- **Linear transformation:** Function T(x) = Ax; invertible if A is invertible.
