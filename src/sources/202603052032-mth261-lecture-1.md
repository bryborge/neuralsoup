# Source: MTH261 - Lecture 1

## tl;dr

> Introduction to linear equations and systems, their graphical and algebraic solutions, and the use of matrices and elementary row operations to analyze and solve them.

## Key Topics

- Linear equations and standard form
- Systems of linear equations
- Solution sets: existence and uniqueness
- Graphical and algebraic solution methods
- Matrix and augmented matrix representations
- Elementary row operations: replacement, interchange, scaling
- Row equivalence and preservation of solutions

## Key Terms & Definitions

- **Linear equation:** An equation of the form $c_1x_1 + c_2x_2 + \cdots + c_nx_n = k$.
- **Linear system:** A collection of one or more linear equations with the same variables.
- **Solution:** A list of values for the variables that makes all equations in a system true.
- **Consistent system:** A system with at least one solution (unique or infinite).
- **Inconsistent system:** A system with no solution.
- **Solution set:** The set of all solutions to a linear system.
- **Coefficient matrix:** A matrix containing only the coefficients of the variables.
- **Augmented matrix:** The coefficient matrix with an extra column for constants.
- **Elementary row operations:** Replacement, interchange, and scaling of rows in a matrix.
- **Row equivalent:** Two matrices related by a sequence of elementary row operations.
