# Source: MTH261 - Lecture 1

## tl;dr

> Introduces the fundamentals of linear systems, exploring how to represent them using matrices and solve them through elimination and elementary row operations.

## Key Topics

- **Linear Equations vs. Nonlinear Equations:** Identifying equations where variables are only to the first power and not multiplied together.
- **Systems of Linear Equations:** Collections of linear equations involving the same set of variables.
- **Solutions and Existence:** Determining if a system has one, none, or infinitely many solutions.
- **Graphical Representation:** Using line intersections to visualize solutions in two-dimensional systems.
- **Matrix Representations:** Converting linear systems into coefficient and augmented matrices for easier manipulation.
- **Elementary Row Operations:** Techniques (Replacement, Interchange, Scaling) used to transform matrices while preserving their solution sets.

## Key Terms & Definitions

- **Linear Equation:** An equation that can be written in the form $c_{1}x_{1} + c_{2}x_{2} + \cdots + c_{n}x_{n} = k$.
- **Consistent System:** A linear system that has at least one solution (either exactly one or infinitely many).
- **Inconsistent System:** A linear system that has no solution.
- **Solution Set:** The set of all possible solutions for a given linear system.
- **Coefficient Matrix:** A matrix containing only the coefficients of the variables from a linear system.
- **Augmented Matrix:** A matrix containing the coefficients of the system plus an additional column for the constants.
- **Row Equivalent:** Two matrices are equivalent if one can be transformed into the other through a sequence of elementary row operations.
- **Replacement:** A row operation where a row is replaced by the sum of itself and a multiple of another row.
- **Interchange:** A row operation where two rows are swapped.
- **Scaling:** A row operation where all entries in a row are multiplied by a non-zero constant.
