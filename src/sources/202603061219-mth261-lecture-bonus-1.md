## Key Terms & Definitions

- **Cramer's Rule:** A method for solving a system of n linear equations with n variables using determinants.
- **Determinant:** A scalar value computed from a square matrix, used to determine invertibility and solve systems.
- **Ai(b):** The matrix formed by replacing the i-th column of A with vector b.
- **Adjugate (adjA):** The transpose of the cofactor matrix of A, used in the formula for the inverse.
- **Inverse of a matrix:** For an invertible matrix A, the matrix A⁻¹ such that AA⁻¹ = I.
- **Cofactor:** The signed minor of an entry in a matrix, used in determinant and adjugate calculations.
