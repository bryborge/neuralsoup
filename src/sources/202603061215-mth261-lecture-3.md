## Key Terms & Definitions

- **Vector:** A matrix with one column; an element of Rⁿ.
- **Linear combination:** A sum of scalar multiples of vectors.
- **Span:** The set of all linear combinations of a set of vectors.
- **Matrix equation:** An equation of the form Ax = b, where A is a matrix and x, b are vectors.
- **Row reduction:** The process of applying row operations to solve matrix or linear systems.
- **Identity matrix:** A square matrix with 1's on the diagonal and 0's elsewhere.
- **Pivot position:** The location of a leading entry in echelon form.
- **Economic equilibrium:** A state where sector outputs balance expenses, modeled by linear systems.
- **Balancing chemical equations:** Using linear algebra to find integer solutions for chemical reaction coefficients.
