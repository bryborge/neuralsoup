## Key Terms & Definitions

- **Standard Matrix:** The unique matrix A such that T(x) = Ax for a linear transformation T.
- **Basis Vector (e₁, e₂, ...):** Columns of the identity matrix used to construct standard matrices.
- **Onto (Surjective):** A transformation T : $\mathbb{R}^n \to \mathbb{R}^m$ is onto if every vector in $\mathbb{R}^m$ is the image of some vector in $\mathbb{R}^n$.
- **One-to-One (Injective):** T is one-to-one if each vector in $\mathbb{R}^m$ is the image of at most one vector in $\mathbb{R}^n$.
- **Pivot Position:** A location in a matrix that corresponds to a leading 1 in row-echelon form, used to determine onto/one-to-one properties.
- **If and Only If:** A logical phrase indicating both directions of implication in a theorem.
