# Neural Soup

My personal knowledge management system.

## Workflow

1.  Capture
    1.  Create a new source file
    2.  Apply the template
    3.  Brain dump
2.  Process (AI-assisted)
    1.  Prompt the AI

        (e.g. "read these lecture notes and suggest 3 atomic zettels. Format your response as checkboxes to paste into my Synthesis Queue.")
3.  Synthesis / Deep Work
    1.  Draft the new zettel file
    2.  Apply the appropriate template (concept, argument, question, etc)
    3.  Write the core concept in your own words
        1.  Try to capture the "what", "why", and "how" (this is often done with more than one zettel)
        2.  Maintain atomicity, prefer conciseness, but use illustrative language as needed
4.  Commit
    1.  Commit (and optionally push) to version control system.

        (e.g. commit message: `Feature [calc-1]: process and synthesize L'Hopital's Rule lecture into zettels`)
